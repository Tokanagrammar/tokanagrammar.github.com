Welcome to Tokanagrammar (pronounced as tok - anagram - mar)

This is a jigsaw-like puzzle game, except each piece is a token from a source file, and the 'complete picture' is the program.

View Tokanagrammar on GitHub

What's in the name?

(I know what you're thinking, but that is not a typo!)

Tok is for token.

anagram is a reference to certain properties that anagram and this game share. That is, this game's objective is to rearrange given tokens to produce programs that are not only syntactically correct but that also produce some expected output.

[gram]mar refers to computer languages' grammars, which players ought to know in order to win the game.

Vision Statement

This is a jigsaw-like puzzle game, except each piece is token from a source file, and the 'complete picture' is the program. The application will be available in both web and desktop application, and if time permits, we look forward to converting it to a mobile app.

Source code
